# Bilder für die Präsentation

Lade hier deine eigenen Bilder ab. Wenn du sie nutzen willst, setze in **`lib/editorialImages.ts`** die Variable **`USE_LOCAL_IMAGES = true`**.

Erwartete Dateinamen:

| Datei       | Verwendung        |
|------------|--------------------|
| hero.jpg   | Hero-Slide         |
| problem.jpg| Problem-Slide      |
| process.jpg| Prozess-Slide      |
| impact.jpg | Wirkung-Slide      |
| contact.jpg| Kontakt-Slide      |

Format: JPG oder PNG, Querformat empfohlen (z. B. 1600×900 px).
